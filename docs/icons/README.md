# 图标方案

这里提供 3 版无文字 SVG 图标方案，适合继续细化为 Android 应用图标。

## 方案 1

- 文件：`concept-1-calendar-clock.svg`
- 方向：日历 + 时钟
- 特点：语义最直接，去掉额外角标后更简洁，适合做最终应用图标

## 方案 2

- 文件：`concept-2-night-shift.svg`
- 方向：夜间加班感的时钟抽象图形
- 特点：更现代、更品牌化，适合偏克制高级的视觉路线

## 方案 3

- 文件：`concept-3-growth-ring.svg`
- 方向：时间圆环 + 收入增长柱形
- 特点：更偏“加班工资上涨”的表达，识别度高

## 使用建议

- 如果你想先求稳，优先选方案 1
- 如果你想更像正式产品图标，优先选方案 2
- 如果你想强调“工资计算”和“增长感”，优先选方案 3
