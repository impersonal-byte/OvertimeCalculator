# v2.2.0

## 变更内容

- 重做每日录入交互，使用居中滑杆替代手写输入，录入加班/调休时更直接、更稳定
- 新增完整数据备份与恢复能力，可在“数据管理”中导出 `.obackup` 备份并通过应用内流程恢复
- 恢复流程新增预览确认、CSV 误选拦截与事务化写入，降低误恢复和数据混杂风险
- CSV 导出继续保留为轻量分享功能，并明确与完整备份/恢复区分
- 收紧 Android 平台自动备份范围，避免把更新会话和节假日缓存这类非关键状态带入恢复结果

## 发布说明

- 本版本在 2.1.0 既有正式版本基础上继续推进，合并了新的结构化录入体验与完整数据备份/恢复闭环，因此版本号提升至 2.2.0
- 备份恢复会优先回到备份中最新的有效业务月份，避免被自动生成的未来配置月份误导
- 发布产物仍为正式签名的通用 APK，可直接用于手动安装与应用内更新
