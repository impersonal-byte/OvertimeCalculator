# 加薪 v1.4.1

本次版本用于验证完整的 GitHub Actions 云端发版链路。

## 更新内容

- 启用双工作流 CI/CD：`Android CI` 与 `Android Release`
- 增加 tag 与应用版本号一致性校验
- 支持优先读取 `docs/releases/<tag>.md` 作为 Release 正文
- Release 工作流先上传 APK 工件，再发布 GitHub Release
- 保持正式签名 APK 作为唯一发布产物

## 说明

- 推送 `main` 只做校验，不创建 Release
- 推送 `v*` tag 才会在云端正式签名并发布 APK
