# OvertimeCalculator v1.0.0

首个公开版本，提供基础的加班记录与工资计算能力。

## 亮点

- 月历形式查看整月加班记录
- 点击日期快速录入加班时长
- 按工作日、休息日、法定节假日自动计算工资倍率
- 支持手动设置时薪
- 支持根据当月已发加班工资反推时薪
- 首页实时显示当月累计加班工资和总时长
- 本地离线保存数据

## 包含内容

- Kotlin + Jetpack Compose 原生 Android 实现
- Material 3 界面
- Room 本地数据库
- 基础单元测试
- 基础 Compose UI 测试

## 当前限制

- 当前为本地单机版本
- 暂无云同步
- 暂无导出 Excel / CSV / PDF
- 暂无通知提醒和桌面组件
- 节假日数据仍会持续完善
