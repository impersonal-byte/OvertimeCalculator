# v2.3.1

## 变更内容

- 调整节假日规则：只把 3 倍工资的法定节假日记为“节假日”，其余法定休息日按“休息日”处理，调休日继续按工作日处理
- 将节假日远程刷新切换为按年度、带 `wage` 语义的数据源，避免把官方放假日全部压平成同一种类型
- 收紧工作日录入滑杆，让 `0h → 6h` 更容易精细选择，同时保留 `-8h` 调休输入能力
- 对历史上已经录入超过 `6h` 的工作日记录保留扩展上限，避免打开编辑器时被静默截断
- 重做设置页细节：主题模式改为紧凑三联分段选择，设置页顶部状态区更统一，备份 / 恢复 / 导出按钮层级更协调

## 发布说明

- 这是基于 `v2.3.0` 的补丁版本，核心目标是完成 Phase 05 的细节体验优化，而不是引入新的主流程能力
- 本次发布同时补齐了阶段执行文档与 roadmap / requirements / state 状态，使仓库中的规划状态与实际代码结果重新一致
- Android 单元测试、调试构建和 AndroidTest 编译已通过；设备侧 UI 验证仍受当前环境无连接设备限制，建议在真机上补做最终观感确认
