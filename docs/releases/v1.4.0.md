# 加薪 v1.4.0

这次是一次中型版本更新，重点补强了两条底层能力：节假日规则更新链路，以及金额计算精度。

## 更新内容

- 节假日热更新改为接入 Timor 中国节假日 API
- 保留应用内置节假日基线数据，远端规则只作为覆盖层，不会冲掉离线兜底
- 新增 DataStore 缓存与 WorkManager 后台静默刷新
- 财务相关计算全面迁移到 `BigDecimal`
- Room 月配置存储从浮点列迁移为十进制字符串，避免金额精度漂移
- 补充 Timor API 映射测试、节假日 JSON 解析测试与 BigDecimal 相关回归测试

## 说明

- 应用仍然支持完全离线使用
- 当 Timor API 未返回某个年份的有效调休/节假日规则时，会自动回退到应用内置基线
- Release 附带 APK，可直接安装，也可作为应用内检查更新的下载源
