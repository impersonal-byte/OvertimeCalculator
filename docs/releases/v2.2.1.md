# v2.2.1

## 变更内容

- 修正 Android release 构建中备份规则 XML 的 lint 问题，恢复正式签名 APK 的自动发版链路
- 保留完整数据备份与恢复能力，可在“数据管理”中导出 `.obackup` 并通过应用内流程恢复
- 恢复流程继续保留预览确认、CSV 误选拦截与事务化写入，降低误恢复和数据混杂风险
- CSV 导出继续保留为轻量分享功能，并明确与完整备份/恢复区分

## 发布说明

- 这是基于 `2.2.0` 发版失败后的补丁版本，主要用于修正 release 配置并确保 GitHub Actions 能成功产出正式签名 APK
- 用户侧 backup/restore 与结构化录入改进保持不变，正式发布版本号提升至 `2.2.1`
- 发布产物仍为正式签名的通用 APK，可直接用于手动安装与应用内更新
