# v2.0.0

## 变更内容

- 完成一次覆盖应用壳层、首页、编辑器、设置模块、更新链路与节假日同步的大规模重构
- 将主界面与设置模块拆分为更清晰的 shell、navigation、section 和 helper 结构，显著降低单文件复杂度
- 引入外观偏好 repository，收口主题与外观设置的读取、状态分发与兼容路径
- 为节假日远端拉取与应用内更新流程抽出独立边界，明确 release check、下载状态与安装引导职责
- 补齐代码库地图、存储边界文档、Room schema 导出、Android UI smoke CI 和新边界的单元测试

## 发布说明

- 这是一次以可维护性、可测试性和后续演进为核心的重大版本升级，因此版本号提升至 2.0.0
- 当前仍保持单模块与手写依赖装配方案；经过本轮重构评估，现阶段暂不引入 Hilt 和多模块拆分
- 发布产物仍为正式签名的通用 APK，可直接用于应用内更新与手动安装
