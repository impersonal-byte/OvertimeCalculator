# v1.6.0

## 变更内容

- 重构首页展示逻辑，保留极简主视图并增加微型月度工时趋势图
- 增强日历高负载色彩层次，提升高时长日期的视觉辨识度
- 重构设置模块，拆分为规则、偏好、数据管理、关于加薪四个二级页面
- 新增外观个性化能力，支持系统默认、浅色、深色、动态取色与多套种子色
- 新增本月 CSV 导出与系统分享链路，支持免存储权限分发文件
- 修正 CSV 导出为后台生成，避免在主线程执行文件 I/O
- 修正 CSV 汇总中的加权总工时计算口径
- 修正主题状态与导出事件的收集方式，减少无效重组和生命周期外收集

## 发布说明

- 本版本聚焦首页与设置模块体验升级，同时补齐数据导出与外观个性化能力
- 发布产物仍为正式签名的通用 APK，可直接用于应用内更新与手动安装
