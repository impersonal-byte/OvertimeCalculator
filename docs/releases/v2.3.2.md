# v2.3.2

## 变更内容

- 调整日期编辑弹窗：保存按钮移动到日期标题右侧，底部操作区移除，关闭弹窗保留为取消路径
- 将日期类型覆盖选项固定为同一排展示，适配 20:9 高分辨率手机屏幕下的紧凑编辑布局
- 优化薪酬与规则页的时薪输入与反推输入，改为分段胶囊输入区 + 对号确认按钮，减少按钮文字占用
- 补充 UI 测试约束，覆盖日期编辑保存入口、日期类型覆盖单排布局，以及薪酬确认入口的图标化展示
- 新增 Phase 06 Material 3 交互系统化升级规划，为后续状态反馈、设计 token、可访问性和 md3E 细化提供执行蓝图

## 发布说明

- 这是基于 `v2.3.1` 的小版本迭代，重点是继续收紧高频编辑与设置表单的交互效率
- 本次发布包含源代码、Android UI 测试和阶段规划文档更新
- Debug 单元测试与 Debug APK 构建已通过；当前环境未连接 Android 设备，设备侧 instrumentation 测试需在真机或模拟器上补跑
