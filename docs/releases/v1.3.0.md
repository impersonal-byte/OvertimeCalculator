# 加薪 v1.3.0

本次更新聚焦于应用内更新链路和首页日历显示稳定性，方便后续直接在真机上验证版本升级体验。

## 更新内容

- 新增设置页“检查更新”，可从 GitHub 最新正式版本检查、下载并安装 APK
- 支持版本比较、下载状态展示，以及未知来源安装授权后的自动继续安装
- 修复首页日历在高分辨率机型上的日期显示裁切问题
- 优化更新状态区右侧加载指示器的布局位置
- 补充更新逻辑的单元测试与界面测试断言

## 适合关注的变化

- 后续版本可直接通过应用内检查更新完成安装
- 首页日历在主流安卓手机上显示更稳定，两位数日期不会再被截断

## 说明

- Release 附带 APK 作为更新源与体验包
- 当前发布仍使用 release APK 分发，未接入应用商店
